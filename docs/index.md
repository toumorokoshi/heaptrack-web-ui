# Heaptrack Web UI Documentation

Welcome to the Heaptrack Web UI documentation. 

This project aims to provide a robust, fast, and entirely client-side web interface for analyzing memory issues generated by `heaptrack`. Instead of relying on a dedicated backend server or local desktop application, users can quickly drag and drop their profiling files directly into the browser to gain deep insights into application memory performance.
